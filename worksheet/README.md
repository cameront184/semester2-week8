
# Worksheet for week 8 - structures and modular programming

- Minerva quiz

## Coding task

The code provided is a simple modular solution for the task.

Do not edit the following files:
- `Makefile` - for compilation
- `shapes.h` - header file with structure and function definitions

You should edit the folllowing:
- `shapes.c` - implement the functions defined in `XXX.h`
- `main.c` - use this to test the functions in `XXX.c`

## Submission
